#### 常用数据结构

##### 链表
> 说明  
> 在链表中，各对象按线性顺序排序。不同于数组的线性序（由数组的下标决定），链表中各对象的顺序是由各对象中的指针决定的。因此，链表可以用来表示动态数组。
>
> 操作  
> 创建列表、判断列表是否为空、插入第一个元素、向列表头部插入元素、查找列表中的元素、删除制定元素、从头至尾显示列表元素
> 
> [java代码](./java/dataStructure/linklist.java)

##### 队列
> 说明  
> 一种先进先出的动态集合。
>
> 操作  
> 创建、插入、查找、删除、显示
